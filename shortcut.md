#shortcut for spacemacs

| 按键                       | 描述                                    |
|:---------------------------|:----------------------------------------|
| C-[                        | evil退出编辑模式                        |
| spc w M window左上角的字母 | 切换窗口                                |
| .点号                      | evil mode重复执行前一次命令             |
| spc b . m-x                | 交换buffer内容                          |
| m-a                        | 光标到行首                              |
| m-e                        | 光标到行尾                              |
| m-d                        | 向后删除单词                            |
| m-w                        | 复制到kill-ring-buffer                  |
| m-m k p                    | 粘贴                                    |
| m-backspace                | 向前删除                                |
| tab                        | markdown模式自动对齐                    |
| C-M-a  / {                 | 上移到函数头部                          |
| C-M-e  / }                 | 下移到函数尾部                          |
| C-@                        | 设置mark标记，可以%跳转或者移动光标多选 |
| spc m g g                  | 跳转到符号定义                          |
| spc m g r                  | 搜索该符号的引用                        |
| m 字母                     | 设置对应字母名的标记                    |
| ·（反引号） 字母           | 跳到字母标记的位置                      |
| ··                         | 跳到上次标记位置                        |
| ‘（单引号）字母           | 跳到字母标记的行首                      |
| C-o                        | 返回到上一次光标位置                    |
| C-M-h                      | 选中整个函数代码                        |
| C-M-f                      | 向前跳转一个语法单位，可类推其他移动    |
| C-l / z z                  | 当前行在可视窗口上中下位置显示          |
